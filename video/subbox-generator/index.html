get your subbox

migrate your youtube subscriptions into a xxd video subbox

  1. get a valid api key. note that you need to create a project for that, and don't forget to enable the "youtube data api v3".
    enter the key in the field below:
  2. go to your youtube subscriptions list directly via the link and scroll to the very bottom (to load everything).
  3. open the developer console. (usually right-click -> inspect -> console)
  4. enter console.log($$("#main-link.channel-link").reduce((str, el)=>str+'\n'+el.href)) into the console.
  5. the returned list of links should look like this:
    https://www.youtube.com/user/channel1
    https://www.youtube.com/channel/UCiD_f0rch4nNeL-tWo
    https://www.youtube.com/user/channel3
    ...

    paste this list into the field below:
  6. click the button below to generate your subbox link
